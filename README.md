# raspi-shared-db-service

Shared MySQL database service for the Raspberry Pi. Runs as a standalone Docker Compose stack; other app stacks on the Pi opt into its network (`shared-db-net`) to consume it.

## Status

Scaffold in progress — see open [GitHub Issues](https://github.com/airlangga07/raspi-shared-db-service/issues) for the full plan.
